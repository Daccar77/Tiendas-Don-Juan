# 🛒 Proyecto: Análisis de Ventas - Alura Store Latam

## 📌 Propósito del Proyecto
Evaluar el desempeño de ventas en ciudades de Latinoamérica para recomendar al Sr. Juan qué tienda cerrar y dónde potenciar su estrategia, utilizando métricas como ingresos, productos vendidos, reseñas y costos de envío.

## 🗂️ Estructura del Proyecto
- 📓 `ChallengerAluraStoreLatam.ipynb`: Análisis completo de datos.
- 📄 `README.md`: Guía explicativa del proyecto.

## 📊 Ejemplos de Gráficos
- 📈 Barras: Ingresos por categoría.
- 🥧 Circular: Envíos por ciudad.
- 🎯 Dispersión: Calificación vs. productos vendidos.

## 🔍 Principales Hallazgos
- Medellín: Mayor volumen de ventas y satisfacción.
- Cartagena: Bajos ingresos; sugerencia de cierre.
- Muebles y Electrodomésticos: Categorías más fuertes.
- Tarjeta de crédito: Método de pago principal.

## 🚀 Instrucciones para Ejecutar
1. Descargar o clonar el repositorio.
2. Abrir el archivo `ChallengerAluraStoreLatam.ipynb` con:
   - Google Colab
   - Jupyter Notebook
   - Visual Studio Code (extensión Jupyter)
3. Ejecutar las celdas en orden para reproducir el análisis completo.
